# Smith Future Roadmap

This document tracks ideas under consideration. Nothing here is committed to a release.

## Project Context (potential Sprint 3)

```bash
smith context .
```

**Purpose:** Generate and persist project metadata such as:

- Language and framework detection
- Build system identification
- Database technologies in use
- Docker usage
- CI/CD presence
- Project modules and structure

This would build on the scan metadata already collected by `AnalyzeProjectTool` and the standardized `ToolResult.metadata` fields introduced in Sprint 2.

**Status:** Not part of Sprint 2. Document only.
